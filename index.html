About this site

This is a plain static site: just a single HTML file served via GitHub Pages, mapped to the domain punxsyborough.com.

Quick editing guide:

  • Each heading (Budget, Council Minutes, etc.) is a <section class="group">.
  • Each document link is a <li class="doc">...</li> entry.
  • Upload PDFs into a docs/ folder in the repo and update the href paths.
